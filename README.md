# 📰 Simple Magazine Website

A basic multi-page website built using **HTML only** to demonstrate fundamental web development concepts.

---

## 📌 Project Description

This project is a simple website that consists of multiple pages:

* 🏠 **Magazine Page**: Main page with sections (Sports & Art)
* 📝 **Register Page**: User registration form
* 🔐 **Login Page**: User login form
* 🧪 **Media Page**: Demonstration of HTML elements (buttons, video, iframe, download)

---

## 🛠️ Technologies Used

* HTML5

---

## 📂 Project Structure

```
📁 project-folder
│── index.html        # Magazine Page
│── register.html     # Register Page
│── login.html        # Login Page
│── media.html        # Media Page
```

---

## 🚀 Features

* Working with **Forms** (Login & Register)
* Using **Tables**
* Displaying **Images & Videos**
* Embedding pages using **iframe**
* File download using `download` attribute
* Using **Ordered & Unordered Lists**
